Laravel CMS

CMS (Content Management System, ou Sistema de Gestão de Conteúdo), feito utilizando o Framework Laravel, banco de dados MySQL, utilizando a integração jeroennoten/Laravel-AdminLTE com o dashboard AdminLTE

Site de classificados feito com HTML, CSS, Bootstrap 3, PHP e MySQL, utilizando conceitos de orientação a objetos. Neste site é possível: ver anúncios, listar anúncios por categoria, preço e estado de conservação, criar conta, publicar, editar e excluir anúncios, visualizar detalhes e fotos do anúncio.
