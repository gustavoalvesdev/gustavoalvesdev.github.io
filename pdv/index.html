Funcionalidades Principais

Cadastros

Este módulo contém os cadastros básicos para o funcionamento do ERP, tais como: categorias, unidades, produtos, contatos, usuários, etc.

Controle de Estoque

Módulo responsável por gerenciar as quantidades existentes de cada produto, entradas, saídas, organização por localização, transferências, ficha razão, etc.

Compras

Controle de Compras através de três processos principais importantes: Solicitação, Cotação e Ordem de Compra.

Controle de Produção (PCP)

O processo produtivo pode ser definido como um conjunto de operações e fases realizadas sucessivamente e de maneira planificada: conformação, polimento, montagem, pintura, estocagem.

Financeiro

O Módulo Financeiro é responsável pelo controle de suas contas a pagar e contas a receber, aprovação das ordens de Compras, Faturamento dos Pedidos, Emissão de NFe, Fluxo de Caixa, etc.

Contábil

No Módulo Contábil temos o plano de contas, o livro diário que servirá como base para emitir os principais relatórios contábeis como Balanço Patrimonial, DRE, etc.

Loja Virtual

Módulo responsável pelas vendas online dos produtos fabricados, isso porque nosso ERP é focado principalmente em fábricas e pequenas indústrias. No módulo de Pedidos, as vendas são feitas pelo atacado, para revendedores e varejistas. Na loja virtual, os focos são as vendas no varejo para clientes finais.
